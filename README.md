# jabir-s-mobile-calc
This is a simple calculator application built using HTML, CSS, and JavaScript. It works offline as a Progressive Web App (PWA) and can be added to a mobile device's home screen for a native app-like experience.

Features

Basic arithmetic operations: Addition, Subtraction, Multiplication, Division.

Clear and backspace functionality.

Works offline using a service worker.

Responsive design for mobile and desktop.

Can be installed as a PWA on mobile devices.
Usage

Open the app in your browser or on your mobile device.

Perform basic calculations using the on-screen buttons.

Add the app to your home screen for quick access.

Files and Structure

/
├── index.html         # Main HTML file
├── style.css          # Stylesheet for the calculator UI
├── service-worker.js  # Service Worker for offline support
├── manifest.json      # Web App Manifest for PWA features

Key Files

index.html: Contains the structure of the calculator.

style.css: Defines the visual design of the calculator.

service-worker.js: Handles caching for offline functionality.

manifest.json: Provides metadata for the PWA.
How to Add to Home Screen

On Android:

Open the app link in Chrome.

Tap the three-dot menu in the top-right corner.

Select "Add to Home Screen."

On iOS:

Open the app link in Safari.

Tap the share icon at the bottom of the screen.

Select "Add to Home Screen."
